# Student Grade Tracker

This project is part of my Python basics assignment.

I built a simple command-line program that works with student data and performs different operations step by step.

## Features

- Cleaned raw student data (fixed names, converted types)
- Converted marks from string format into a proper list
- Checked if student names are valid
- Displayed formatted student profile cards
- Calculated total and average marks
- Found highest and lowest scoring subjects
- Added new subjects using user input
- Generated a class performance summary (pass/fail, topper, class average)
- Performed some basic string operations on a sample paragraph

## Concepts Practiced

- for loops and while loops  
- if-else conditions  
- lists and dictionaries  
- string methods  
- functions  
- basic error handling
